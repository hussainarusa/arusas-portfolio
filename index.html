Berry to Bean

The application is an immersive coffee website that allows users to seamlessly search for and favorite different coffee blends. It features a comprehensive CRUD system for managing coffee-related data. The design reflects the company's brand vision, resembling an authentic coffee retailer platform. The use of HTML, JavaScript, CSS, and Bootstrap ensures a dynamic and visually captivating user experience, showcasing a fusion of technical acumen and creative flair.

Kyte Rescue

Kyte Rescue aims to simplify animal fostering and address shelter overcrowding. This full-stack web application leverages Java, JavaScript, CSS, and HTML. Key features include an intuitive user interface for fostering, CRUD functionalities for pet management, and real-time access to adoptable animals through the Pet Finder OAuth API.

TIMOA Flicks

Timoa Flicks is a dedicated application tailored for movie enthusiasts, providing a platform to effortlessly collect and favorite movies online. With Timoa Flicks, users can explore a vast array of movies, save their favorites, and assign personal ratings to create a curated collection. The application offers a user-friendly search functionality, enabling users to easily discover and bookmark movies of interest.

Shadow Trade

Shadow Trade is a dynamic full-stack application that seamlessly integrates both front-end and back-end functionalities to provide users with a comprehensive online marketplace experience. Similar to Craigslist, users can easily create and post ads, fostering a diverse and vibrant marketplace. The robust back-end database ensures a rich collection of listings, enhancing the platform's usability. Beyond its functionality, the application boasts an engaging theme and design, creating an immersive user interface.

Weather Map
